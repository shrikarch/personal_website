Information System using Dart and MongoDB
April 2016 | Team Size: 2

Dart AngularJs HTML/CSS MongoDB PlatformIndependent
  • A RESTful Mobile information management system based on Dart (a programming language introduced by Google)
  • Used AngularJs for front-end data manipulation and data-binding.
  • Supported by MongoDB as a back end, for it's scalability.

  • The customer's end was available on handheld devices
  • The kitchen end was available for monitors.
  • Order's placed by customers would update real-time on the kitchen end.
  • Table numbers were passed over by GET requests. Other information was passed by POST request.
