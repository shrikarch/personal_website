Wireless Sensor Network using Python to find the network backbone.
Spring 2016 | Team Size: Solo
Published on Devpost

Python NetworkX RandomGeometricGraphs SensorNetworks Algorithms

Created Random Geometric Graphs on a unit plane/disc using Python's NetworkX library. Then a largest possible subgraph was selected out of the generated network graph. Smallest Last graph coloring algorithm was used next to find two longest continuous bipartite graphs - Longest backbones.
The objective of this project was to simulate the optimum path of connectivity for a bunch of sensorts, randomly present in a planar environment.

Smart Mirror at HackDFW
April 2016 | Team Size: 5
Published on Devpost

HackDFW16 MeteorJs VoiceOperated HTML/CSS RaspberryPi

A small scale project made within 24 hours at HackDFW that earned us an opportunity to enter an accelerator program for start-ups.
The whole mirror was made from merely a few things: Used cereal boxes, a wood frame, an acrylic sheet with reflective coating, an LED monitor and a Raspberry Pie!
The interface was designed in HTML5 and was based on meteor.js with multiple APIs used to serve the data like live stocks, emails, tweets and weather and was controlled by voice.
JavaScript library 'annyang!' was used for voice recognition.

Data and Visualization, clustering, association mining and classification of a graduation rate data set
April 2016 | Team Size: Solo

R DataMining

-The Project was done using R in RStudio.
-Using the data sets we have to help increase U.S. high school graduation rate from 81.4% in 2013 to 90% by the year 2020
-The major challenges were understanding the given data, cleaning the data, visualization of the data and finding the relationships between different variables of the data.
-Different Classification models like CART, random forest, conditional inference tree, C4.5 algorithm, linear support vector machines were applied to the dataset.After creating classification models, it was assessed by various techniques like bootstrap validation, cross validation etc.
-Also studied and implemented the association rules between two different data sets.
-In addition to that, clustering analysis using different methods like k-means, hierarchical clustering, dbscan clustering was performed on the set.
